The **AI Financial Assistant** is a full-stack financial analysis system that processes bank statements, categorizes transactions using an LLM (Groq API), analyzes spending trends, and answers natural-language financial questions.  
It supports **PDF, Excel, and CSV** uploads and provides a clean, modern interface with login, file upload, and chat functionality.
